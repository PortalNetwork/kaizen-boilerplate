### Install flow

#### Clone repo

```
git clone https://github.com/PortalNetwork/kaizen-boilerplate.git
cd kaizen-boilerplate
```

#### Start repo
install packages
```
npm install
```

#### develope 
```
npm run dev
```

#### test smart contracts
```
npm run test-contracts
```

#### compile smart contracts
```
npm run compile-contracts
```
